v9 调整说明
⑯ 选中 tab 始终完整可见：tab 过多时，.tabs 改为横向滚动（overflow-x:auto，滚动条隐藏）。选中的 tab 用 scrollIntoView({inline:'nearest'}) 自动滚到可见区——无论选中的是「收藏」还是其他，都完整显示不被裁切。落地到 React 时用 useEffect 监听 activeTab 变化 + tab ref 调 scrollIntoView。点击 tab 后 requestAnimationFrame 触发滚动，保证 class 更新后再滚。
⑰ list 顶部加边距：.list 加 padding-top:6px，第一个 item 不再紧贴 tab 行的 border-bottom 分割线。
v8 历史（⑬-⑮）：收藏标记全局显示 / 标签按钮对齐头像 / item hover 信息与代码一致。
v7 历史（⑨-⑫）：标题后不显示文件夹 / 标签合并单按钮 / 展开/折叠只图标 / Tab 去图标+顺序+去未读点。
v6 历史（⑦-⑧）：标签前加 tag 图标 / Tab 图标+文字+溢出更多+分隔线。
v5 历史（①-⑥）：大厅在 Tab 之上 / 项目完整路径 / 标签色点 / 训练营最近 Tab / 展开折叠右侧 / 单 chevron。
置顶逻辑不变：对话置顶=最近 Tab + 当前 Tab 子目录双重置顶（pushpin 标识）；项目置顶=只项目 Tab（项目名染 accent + pushpin）。排序：最近按时间，其他 Tab 字典序。
